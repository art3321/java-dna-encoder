# java-dna-encoder
# DNA Bit Compressor  A CLI Java tool that compresses and decompresses DNA sequences (A, C, G, T) by packing 4 nucleotides into a single byte using bitwise operations.  ## Commands * `comp [DNA]` — Encodes DNA to HEX bytes. * `decomp [count] [bytes]` — Decodes HEX bytes back to DNA. * `about` — Student info.
